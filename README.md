# AI-ассистент для подготовки к экзаменам

Это техническая документация для MVP проекта AI-ассистента для подготовки к ОГЭ и ЕГЭ.

## Обзор проекта

Проект направлен на создание веб-приложения, которое будет помогать школьникам 8-11 классов находить пробелы в знаниях, объяснять ошибки и строить персональный путь к результату. Приложение будет представлять собой адаптивный сайт с темной и светлой темами, вдохновленными GitHub, и желтыми акцентами.

## Архитектура и Технологии

- **Тип приложения:** Веб-приложение (адаптивный сайт).
- **Архитектура:** Serverless, с ограничением по оперативной памяти до 512 МБ и объемом базы данных до 1 ГБ.
- **API:** Использование Gemini API для реализации функций AI-ассистента.
- **Пользовательский функционал:** Регистрация и вход по email, настройки пользователя, дэшборд для администраторов.
- **Конфигурация:** Переменные окружения через файл `.env`.
- **Контроль версий:** Управление исключениями файлов с помощью `.gitignore`.

## Запуск проекта (предполагаемые шаги)

1.  **Клонирование репозитория:**
    ```bash
    git clone [ссылка_на_репозиторий]
    cd [название_проекта]
    ```
2.  **Установка зависимостей:**
    ```bash
    # Пример для Node.js/npm
    npm install
    # Пример для Python/pip
    pip install -r requirements.txt
    ```
3.  **Настройка переменных окружения:**
    Создайте файл `.env` в корневой директории проекта и добавьте необходимые переменные (например, ключ для Gemini API).
    ```
    GEMINI_API_KEY=YOUR_API_KEY
    # Другие необходимые переменные
    ```
4.  **Запуск приложения:**
    ```bash
    # Пример для локального запуска
    npm start
    # или
    python app.py
    ```
