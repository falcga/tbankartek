# AI-ассистент для подготовки к экзаменам

Веб-приложение для подготовки к ОГЭ и ЕГЭ с AI-объяснением ошибок

## Возможности

- регистрация и вход по email
- интерактивный тренажёр с вопросами
- AI-объяснение каждой ошибки через Gemini API
- персональный план подготовки
- тёмная и светлая тема
- админ-панель

## Установка

```
pip install -r requirements.txt
```

Добавьте GEMINI_API_KEY в .env

## Запуск

```
python app.py
```

## Технологии

- Python + Flask
- SQLite
- Gemini 2.0 Flash API
- HTML + CSS + Vanilla JS
